# نظام إدارة المخزون والمبيعات

## وصف المشروع
تطبيق ويب ديناميكي مصمم لحل مشكلة تتبع المنتجات والكميات المتاحة داخل المخازن والمستودعات لشركات المبيعات. تم بناء النظام الخلفي وتأمين العمليات باستخدام لغة PHP وقاعدة بيانات MySQL، وتنسيق الواجهات ورسائل النظام بالكامل باستخدام Bootstrap 5 ليكون متوافقاً مع الشاشات المختلفة.

## فكرة المشروع
توفير لوحة تحكم (Dashboard) شاملة وخاصة بالمسؤول (Admin) تتيح له إدارة مخزون شركة مبيعات بشكل فوري وآمن، مع تطبيق مستويات الصلاحيات والتحقق من صحة البيانات المدخلة قبل الحفظ.

## الميزات البرمجية المطبقة في النظام
* **لوحة تحكم كاملة للمسؤول (Admin Dashboard):** لمراقبة واستعراض حالة المخزن والوصول السريع للأقسام.
* **تطبيق العمليات الأساسية الأربعة (CRUD):** إضافة منتج جديد، استعراض القائمة، تعديل بيانات المنتج، والحذف الفوري للمنتج وصورته من السيرفر.
* **نظام حماية الجلسات والأمان:** إدارة الدخول والخروج باستخدام `Sessions` وحماية الصفحات من الدخول غير المصرح به، وتشفير كلمات المرور في قاعدة البيانات عبر `password_hash`.
* **رفع الصور وعرضها:** رفع صور المنتجات ديناميكياً مع التحقق من الامتدادات المسموحة (Validation) لضمان أمان السيرفر.
* **تنسيق مرن:** عرض تنبيهات ورسائل خطأ وتأكيد منسقة وملونة للمستخدم بناءً على العمليات.

## طريقة تشغيل المشروع
1. قم بتحميل ملفات المشروع من المستودع أو عمل Clone له.
2. انقل مجلد المشروع `inventory_project` إلى مسار السيرفر المحلي لديك: `C:\xampp\htdocs\`.
3. قم بفتح لوحة تحكم `phpMyAdmin` وأنشئ قاعدة بيانات جديدة باسم `inventory_db`.
4. قم بعمل استيراد (Import) لملف قاعدة البيانات المرفق مع المشروع باسم `inventory_db.sql`.
5. لتشغيل النظام، توجه عبر المتصفح إلى الرابط التالي: `http://localhost/inventory_project/login.php`.

| :--- | :--- | :--- | :--- |
| مدير النظام (Admin) | admin | admin | كامل الصلاحيات (الوصول للوحة التحكم، وعمليات الـ CRUD كاملة) |
